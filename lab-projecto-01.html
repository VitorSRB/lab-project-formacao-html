História da WEB

Voltar

A história da web teve início com os primeiros computadores eletrônicos e sua evolução ao longo do tempo. Os primeiros computadores, como o ENIAC e o Colossus, surgiram durante a Segunda Guerra Mundial, utilizando tecnologia mecânica e eletrônica para realizar cálculos complexos. A partir daí, grandes avanços foram feitos por nomes como Alan Turing, Konrad Zuse e John W. Mauchly, impulsionando o desenvolvimento de computadores eletrônicos programáveis.

Na década de 1950, os computadores eletrônicos evoluíram, tornando-se mais acessíveis e comercialmente viáveis. O IBM 650 foi um marco importante, sendo um dos primeiros computadores de grande escala a ser amplamente adotado.

Nos anos 1960 e 1970, surgiram várias contribuições significativas, incluindo o desenvolvimento da interface gráfica do usuário por Douglas Engelbart e os protocolos de comunicação de rede TCP/IP por Vint Cerf e Robert E. Kahn, que são considerados os pais da internet.

Na década de 1980, Bill Gates e Paul Allen fundaram a Microsoft, desempenhando um papel crucial na popularização dos computadores pessoais e do software.

Em 1989, Tim Berners-Lee criou a World Wide Web, revolucionando a forma como compartilhamos e acessamos informações na internet.

Esses são apenas alguns dos principais nomes e eventos ao longo da história da web. Cada avanço tecnológico contribuiu para a criação da internet como a conhecemos hoje, possibilitando a conectividade global e transformando a maneira como vivemos, trabalhamos e nos comunicamos.

O que são Clientes

Voltar

Os clientes, em relação à web, são os softwares ou aplicativos utilizados pelos usuários para acessar e interagir com os serviços disponíveis na internet. Eles são responsáveis por solicitar e exibir conteúdos da web, como páginas da web, aplicativos web, vídeos, imagens, entre outros.

Alguns dos principais navegadores utilizados no lado do cliente são:

  1. Google Chrome: Desenvolvido pelo Google, o Chrome é um navegador amplamente utilizado em todo o mundo, conhecido por sua velocidade, recursos avançados e compatibilidade.
  2. Mozilla Firefox: O Firefox é um navegador de código aberto desenvolvido pela Mozilla Foundation, que prioriza a privacidade e a segurança dos usuários.
  3. Safari: Desenvolvido pela Apple, o Safari é o navegador padrão nos dispositivos iOS e macOS, oferecendo integração com os produtos da Apple e uma experiência otimizada para esses sistemas.
  4. Microsoft Edge: O Edge é o navegador desenvolvido pela Microsoft, que substituiu o Internet Explorer. Ele oferece suporte a recursos modernos da web e é compatível com os sistemas operacionais Windows 10 e macOS.

Aplicações web são softwares acessados através de um navegador, onde os recursos e funcionalidades são executados no servidor e entregues ao cliente por meio da interface do navegador. As aplicações web não precisam ser instaladas no dispositivo do usuário e podem ser acessadas a partir de diferentes dispositivos.

Já os aplicativos são softwares desenvolvidos especificamente para serem instalados em dispositivos, como smartphones e tablets. Eles podem ser baixados e instalados diretamente no dispositivo do usuário, aproveitando os recursos e funcionalidades do sistema operacional do dispositivo.

No contexto dos dispositivos móveis, o lado do cliente desempenha um papel fundamental. Os dispositivos móveis, como smartphones e tablets, possuem recursos de hardware e sistemas operacionais que permitem a execução de aplicativos móveis nativos e a interação com aplicativos web através dos navegadores. Os navegadores móveis, como o Chrome e o Safari, são utilizados para acessar sites e aplicativos web em dispositivos móveis, proporcionando uma experiência de navegação adequada para telas menores e interfaces sensíveis ao toque. Além disso, os dispositivos móveis também têm acesso a recursos específicos, como sensores de localização, câmeras e notificações, que podem ser explorados pelas aplicações móveis para oferecer funcionalidades adicionais e uma experiência personalizada aos usuários.

O que são Servers?

Voltar

Os servers são computadores ou sistemas responsáveis por fornecer serviços, recursos ou armazentamento de dados a outros dispositivos, conhecidos como clientes. Eles são equipados com hardwares e softwares específicos para gerenciar e atender às solicitações dos clientes.

Os hardwares utilizados em servidores podem incluir processadores potentes, grande capacidade de armazenamento de dados, memória RAM e conexões de rede de alta velocidade. Os softwares em servidores incluem sistemas operacionais de servidor, como o Linux ou o Windows Server, além de aplicativos e serviços especializados, como servidores de banco de dados, servidores de e-mail, servidores web, entre outros.

Existem diferentes tipos de servidores, cada um projetado para um propósito específico. Alguns exemplos incluem:

  1. Servidores de arquivos: Responsáveis pelo armazenamento e compartilhamento de arquivos em rede.
  2. Servidores de banco de dados: Utilizados para armazenar e gerenciar grandes volumes de dados de forma eficiente.
  3. Servidores de e-mail: Responsáveis pelo envio, recebimento e armazenamento de e-mails.
  4. Servidores web: Utilizados para hospedar sites e disponibilizá-los para acesso através da internet.

Os servers também desempenham um papel importante como hospedagem de sites. Nesse contexto, um servidor é configurado para armazenar os arquivos de um site e disponibilizá-los para acesso dos visitantes da web. O servidor responde às solicitações dos navegadores dos usuários, enviando as páginas do site para visualização.

Em resumo, os servers são computadores ou sistemas que fornecem serviços, recursos ou armazenamento para outros dispositivos, utilizando hardwares e softwares especializados. Eles desempenham várias funções, como hospedar sites, gerenciar bancos de dados, armazenar arquivos e fornecer serviços de rede.

Linguagens de programação

Voltar

As linguagens de programação server side são utilizadas no servidor para processar e gerar dinamicamente as páginas da web antes de enviá-las para o cliente. Exemplos de linguagens server side incluem PHP, Python, Ruby e Java. Essas linguagens são responsáveis por interagir com bancos de dados, processar formulários e executar lógica de negócios no lado do servidor.

As linguagens de programação client side são executadas no navegador do cliente, ou seja, no dispositivo do usuário. Elas são responsáveis por manipular a interface do usuário, realizar validações e interações dinâmicas. Exemplos de linguagens client side são JavaScript, HTML e CSS. O JavaScript é a linguagem principal para programação client side, permitindo a criação de interatividade e dinamismo nas páginas da web.

O HTML (HyperText Markup Language) é a linguagem de marcação utilizada para estruturar e exibir conteúdo em uma página da web. É responsável por definir a estrutura do documento, como parágrafos, títulos, links, imagens e outros elementos. O HTML utiliza tags para marcar e estruturar o conteúdo, permitindo que os navegadores interpretem e exibam corretamente as informações.

Introdução ao HTML na prática

Voltar

Primeiramente é importante ressaltar quais serão as ferramentas utilizadas no desenvolvimento do aprendizado em HTML. Apesar de podermos utilizar apenas um editor de textos simples, podemos utilizar uma IDE, Integrarted Development Environment, que é uma ferramenta de desenvolvimento para editar códigos. Neste caso iremos utilizar o Visual Studio Code

Uma dica importante é utilizarmos o Inspetor de elementos do navegador para poder verificar o comportamento das páginas que escrevemos dentro do navegador, como também entender quais métodos de escrita HTML são utilizados por outros sites. Essa ferramenta pode ser acessada com Ctrl + Shift + l ou simplesmente teclando F12.

Começando iremos entender qual é a estrutura mais básica do documento HTML:

  • Html: Essa é a estrutura principal, dentro dela estão todas as outras, como o caso da tag head e body.
  • Head: é o espaço do documento HTML que contém informações de identificação do documento, metadados, e principalmente o nome do documento que será exibido na aba do navegador.
  • Body: É a parte do documento HTML, que contém toda a estrutura da página que é exibida no navegador, onde de fato o client side consegue navegar e utilizar os componentes da página web.

Tags e seus atributos

Voltar

As tags HTML são elementos utilizados para marcar e estruturar o conteúdo de uma página da web. Elas fornecem instruções aos navegadores sobre como exibir e interpretar o conteúdo.

Cada tag HTML é representada por um nome entre os símbolos "<" e ">". Existem tags de abertura, como "<p>", que indicam o início de um elemento, e tags de fechamento, como "</p>", que indicam o fim do elemento. Além disso, algumas tags não possuem conteúdo entre elas, sendo chamadas de tags vazias ou auto-fecháveis, como "<img />".

Além das tags, as tags HTML também podem conter atributos. Os atributos são adicionados às tags para fornecer informações adicionais sobre o elemento ou modificar o seu comportamento. Os atributos são compostos por um nome e um valor, separados por um sinal de igual (=). Por exemplo, o atributo "href" na tag "<a>" é usado para especificar o destino de um link.

Exemplo de uma tag HTML com atributos:

<a href="https://www.exemplo.com" target="_blank">Link para Exemplo</a>

Neste exemplo, a tag "<a>" é utilizada para criar um link. O atributo "href" especifica o URL de destino, enquanto o atributo "target" define que o link deve ser aberto em uma nova janela do navegador.

Em resumo, as tags HTML são elementos utilizados para marcar e estruturar o conteúdo de uma página da web, e os atributos fornecem informações adicionais ou modificam o comportamento dos elementos.
